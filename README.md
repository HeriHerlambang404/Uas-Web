- Deskripsi
  1. Program E - Artikel (Program penyebaran/membaca artikel berbasis Website)
  - Yang dapat di akses 2 pengguna yaitu Pembaca (User) & Admin (Pengelola)
  - Di buat menggunakan PHP, HTML, MYSQL, INFINITYFREE (HOSTING)
  - Konsep Menggunakan OOP (Class Database, Modular System)
  - Desain Mobile First & Tailwind CSS, Bootstrap 5. (Support jika di buka di handphone & Detskop)
  - Fitur yang ada di user
    1. Register
    2. Login/Logout
    3. Dashboard (Home)
    4. Daftar Artikel (Akses artikel yang dapat di baca, mencari artikel, pagination)
    5. My Profil (Cek Informasi profil & ganti password)
  - Fitur yanga ada di administrator
    1. Register
    2. Login/Logout
    3. Dashboard (Home)
    4. Daftar Artikel (Akses artikel yang dapat di baca, mencari artikel, Menambah Artikel, Menghapus Artikel, Mengubah Artikel, pagination)
    5. Tambah User (Tambah Pengguna)
    6. My Profil (Cek Informasi profil & ganti password)
